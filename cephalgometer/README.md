# Cephalgometer

"Confirms headache"

Firmware for indoor weather station with:
- BME280:
  - temperature
  - humidity
  - atmospheric pressure
- MH-Z14A CO<sub>2</sub> sensor
- PCD8544-driven 84x48 display
